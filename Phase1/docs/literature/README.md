# Literature Review Workspace

The assignment explicitly says the literature review should **not** be generated with LLM tools.

Use this folder only to organize your own manual reading notes.

Suggested process:

1. Search Google Scholar and IEEE Xplore for the last 3 years.
2. Pick 5 papers focused on LLM-based test generation.
3. Fill `paper_matrix.csv` manually.
4. Write the literature review section yourself in the IEEE report.
