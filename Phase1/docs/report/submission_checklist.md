# Phase 1 Submission Checklist

Use this checklist after the technical work is complete and before uploading the report to Ninova.

- Replace every Java author header placeholder with the real student names and IDs.
- Fill the `Acknowledgment` section in `phase1_report_draft.md` with the GitHub repository URL.
- Fill the group workload distribution in the `Acknowledgment` section.
- Transfer the final text into the official IEEE journal template.
- Verify that the report does not embed long code listings; reference the repository instead.
- Verify that the repository commit history still tells the step-by-step Phase 1 story.
- Re-run the main Phase 1 scripts if any generated code or tests changed after report writing.
