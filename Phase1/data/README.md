# Data Folder

This folder organizes dataset inputs and prompt selection artifacts for Phase 1.

- `raw/`: keep downloaded dataset files here. Raw compressed files are ignored by Git.
- `selected_prompts.csv`: the 30-task starter selection used by the scaffold.
- `selection_criteria.md`: explains why these tasks were chosen and how difficulty was assigned.

The official HumanEval-X Java dataset file expected by the scripts is:

- `raw/humaneval_java.jsonl.gz`
