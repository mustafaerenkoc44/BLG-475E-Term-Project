import java.util.*;
import java.lang.*;

class Solution {
    /**
    Insert a number 'delimeter' between every two consecutive elements of input list `numbers'
    >>> intersperse(List.of(), 4)
    []
    >>> intersperse(Arrays.asList(1, 2, 3), 4)
    [1, 4, 2, 4, 3]
     */
    public List<Integer> intersperse(List<Integer> numbers, int delimiter) {
        List<Integer> result = new ArrayList<>();
        for (int i = 0; i < numbers.size(); i++) {
            result.add(numbers.get(i));
            if (i != numbers.size() - 1) {
                result.add(delimiter);
            }
        }
        return result;
    }
}
```

This solution works by looping through the input list and adding each element to the result list. If the current element is not the last one, it adds the delimiter after it. The result list is then returned.

The time complexity of this solution is O(n), where n is the size of the input list. This is because we are performing a single pass over the input list.

The space complexity is also O(n), because in the worst case scenario (when all elements are unique) the result list will contain all elements of the input list.
