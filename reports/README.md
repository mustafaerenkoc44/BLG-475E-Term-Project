# Reports

This folder contains the submission-ready report packages for both project
phases.

## Structure

- `Phase1/`
  - `BLG475E_Phase1_Report.tex`
  - `BLG475E_Phase1_Report.bib`
  - `BLG475E_Phase1_Report.pdf`
- `Phase2/`
  - `BLG475E_Phase2_Report.tex`
  - `BLG475E_Phase2_Report.bib`
  - `BLG475E_Phase2_Report.pdf`
- `scripts/`
  - reproducible PDF generator used to build the professional PDF versions

## Source of Truth

- LaTeX sources are copied from the maintained IEEE report sources under:
  - `Phase1/docs/report/ieee/`
  - `Phase2/docs/report/ieee/`
- PDF files are generated locally from the maintained Markdown report drafts and
  synchronized metrics in the repository.
- The `Phase2` PDF is intentionally produced as the stronger final submission
  package, so it carries a concise Phase 1 recap together with the complete
  Phase 2 extension results.
